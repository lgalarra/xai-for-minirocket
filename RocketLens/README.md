# RocketLens: Interactive Visual Analytics for Explaining MiniRocket Time Series Classifiers

A live demo can be found at https://rocketlens.webs.upv.es/.

## Deploying the demo locally.

- Clone this repository
- Make sure that you have Node.js installed on your server and a Python environment that can execute the provided augmented version of MiniROCKET (see the README in the root of this repository)
- Using a command line tool locate yourself in the root folder of the repository and run:
  
```$ node  RocketLens/server.js``` 

The application will run on http://localhost:3000. The demo depends on the extended version of MiniROCKET included in this repository, so make sure you can execute MiniROCKET from the shell that launches 
the demo.
